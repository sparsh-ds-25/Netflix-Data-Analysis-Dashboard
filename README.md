# Netflix Data Analysis & Dashboard Project 📺📊

## Project Overview

This project focuses on analyzing Netflix content data using Python and visualizing insights through an interactive Power BI dashboard. The dataset was taken from Kaggle and used to explore content trends, audience ratings, genre distribution, release patterns, and country-wise contributions.

The project combines Exploratory Data Analysis (EDA) with Business Intelligence techniques to transform raw data into meaningful visual insights.
